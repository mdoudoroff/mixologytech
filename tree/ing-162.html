    framboise liqueur

    • liqueurs /
    • liqueurs, fruit-flavored /
    • raspberry liqueur /
    • framboise liqueur

    description

    Framboise liqueur is generic for sweet French raspberry liqueur, not to be confused with framboise eau de vie or framboise lambic (a beer). Care must be taken when preparing a recipe or calling for a drink to note WHICH version of Framboise is really called for, since often either one will be listed merely as "framboise".

    flavors
    sweet, fruit, raspberry, berries
    substitute
    other raspberry liqueur or crème de cassis (imperfect substitute)
    places (country/region of production)
    France
