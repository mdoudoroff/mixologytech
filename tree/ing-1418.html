    honey-flavored liqueur

    • liqueurs /
    • liqueurs, flower-flavored /
    • honey-flavored liqueur

    synonyms

    honey liqueur

    description

    Honey-flavored liqueur is a generic category for liqueurs incorporating honey—generally as the sweetening agent and primary flavoring—with neutral spirits and herbs and spices. This category has enormous latitude for diversity, so discretion is required.

    flavors
    floral, sweet, honey, spice
    substitute
    Home-made
    places (country/region of production)
    Kentucky (United States), United States, Italy, Germany, Estonia

    examples

    • Bärenjäger Honey Liqueur
    • Christian Brothers Honey Liqueur
    • Evan Williams Honey Reserve
    • Koval Organic Chrysanthemum Honey
    • Liviko Honey Liqueur
    • Nonino Gioiello Castango
